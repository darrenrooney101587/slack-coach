  <files>app/fireflies.py</files>
  <action>
Create app/fireflies.py with two public functions:

1. `verify_fireflies_signature(secret: str, raw_body: bytes, signature_header: str) -> bool`
   - Returns False immediately if secret or signature_header is falsy
   - Computes `hmac.new(secret.encode("utf-8"), raw_body, hashlib.sha256).hexdigest()`
   - Compares using `hmac.compare_digest` to prevent timing attacks
   - Note: Fireflies signs the raw payload directly (no `v0:timestamp:` prefix unlike Slack)
   - Note: signature_header may arrive as raw hex OR with a `sha256=` prefix — strip any such prefix before comparing

2. `fetch_transcript(meeting_id: str, api_key: str) -> dict`
   - POSTs to `https://api.fireflies.ai/graphql` with `Authorization: Bearer {api_key}` header
   - Uses the GraphQL query requesting: id, title, transcript_url, participants, summary { overview, action_items, bullet_gist }
   - Always passes `timeout=10` to requests.post
   - Calls `response.raise_for_status()` before parsing
   - Returns `response.json().get("data", {}).get("transcript", {})` — returns empty dict on missing path, never raises KeyError

Module-level constant: `FIREFLIES_API_URL = "https://api.fireflies.ai/graphql"`

Imports needed: os, hmac, hashlib, requests

Do NOT import from app modules — this module has no internal dependencies.

  <name>Task 2: Create app/formatter.py — pure Block Kit formatter</name>

  <action>
Create app/formatter.py with one public function:

`format_recap(transcript: dict) -> list`

Implementation requirements:
- Extract `title` with fallback: `transcript.get("title") or "Meeting Recap"`
- Extract `transcript_url`: `transcript.get("transcript_url", "")`
- Extract `summary` dict: `transcript.get("summary") or {}`
- Extract `overview`: `summary.get("overview") or summary.get("bullet_gist") or ""`
- Extract `action_items_raw`: `summary.get("action_items") or ""`
- Handle action_items_raw defensively:
  - If `isinstance(action_items_raw, list)`: join with `"\n".join(f"- {a}" for a in action_items_raw if a)`
  - Otherwise: `str(action_items_raw).strip()`
- Truncate overview to 2900 chars: `overview[:2900]`
- Truncate action_items_text to 2900 chars: `action_items_text[:2900]`
- Truncate title to 150 chars for the header block plain_text field

Build blocks list in this order:
1. Header block: `{"type": "header", "text": {"type": "plain_text", "text": title[:150]}}`
2. Divider block
3. Summary section (only if overview is non-empty): `{"type": "section", "text": {"type": "mrkdwn", "text": f"*Summary*\n{overview[:2900]}"}}`
4. Action items section (only if action_items_text is non-empty): `{"type": "section", "text": {"type": "mrkdwn", "text": f"*Action Items*\n{action_items_text[:2900]}"}}`
5. Divider block
6. Transcript link section (only if transcript_url is non-empty): `{"type": "section", "text": {"type": "mrkdwn", "text": f"<{transcript_url}|View full transcript>"}}`

Return the blocks list. No I/O, no side effects, no imports from other app modules.

Imports needed: none (pure Python)
