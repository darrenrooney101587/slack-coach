  <action>
Create tests/test_formatter.py. Use pytest with function-based tests and plain assert statements. No mocking needed — format_recap has no external dependencies.

Test cases:

1. `test_empty_dict_returns_list_without_error`
   - Call format_recap({})
   - Assert result is a list
   - Assert len(result) >= 2 (at minimum header + divider)

2. `test_full_transcript_contains_all_sections`
   - Input: title, transcript_url, summary.overview, summary.action_items as list ["Do A", "Do B"]
   - Assert one block has type="header" and text.text == "My Meeting"
   - Assert one mrkdwn block contains "Summary"
   - Assert one mrkdwn block contains "Action Items"
   - Assert one mrkdwn block text contains "- Do A"
   - Assert one mrkdwn block text contains "- Do B"
   - Assert one mrkdwn block text contains "example.com"

3. `test_action_items_as_string_preserved`
   - Input: summary.action_items = "First item\nSecond item"
   - Assert a mrkdwn block text contains "First item"
   - Assert the same block text contains "Second item"

4. `test_null_action_items_omitted`
   - Input: summary.action_items = None (summary dict present but action_items is None)
   - Collect all mrkdwn block texts
   - Assert none contain "Action Items"

5. `test_long_overview_truncated`
   - Input: summary.overview = "x" * 3000
   - Find the mrkdwn block containing "Summary"
   - Assert len(block["text"]["text"]) <= 3000

6. `test_missing_transcript_url_omits_link_section`
   - Input: transcript with summary but no transcript_url key
   - Assert no block text contains "|View full transcript>"

7. `test_empty_transcript_url_omits_link_section`
   - Input: transcript_url = ""
   - Assert no block text contains "|View full transcript>"

8. `test_long_title_truncated_in_header`
   - Input: title = "A" * 200
   - Find the header block
   - Assert len(header_block["text"]["text"]) <= 150

9. `test_missing_title_uses_fallback`
   - Input: {} (no title key)
   - Find the header block
   - Assert header_block["text"]["text"] == "Meeting Recap"

Helper: define a small `_find_blocks(blocks, type_)` helper or use list comprehension inline to locate blocks by type.

conftest.py already adds project root to sys.path, so import with: `from formatter import format_recap`
